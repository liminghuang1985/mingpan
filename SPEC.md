# 命盘 App - 产品需求文档

## 1. 项目概述

### 1.1 项目名称
**命盘** - 传统命理可视化工具

### 1.2 核心功能
通过出生年月日时分，生成动态旋转的命盘可视化，展示八字、大运、流年等信息，带有精美的动画效果。

### 1.3 目标平台
- **网页端**: React + TypeScript + TailwindCSS
- **安卓端**: Flutter（跨平台）
- **计算核心**: 纯 TypeScript 算法

## 2. 功能需求

### 2.1 输入模块
| 功能 | 描述 |
|------|------|
| 出生日期 | 年、月、日（支持农历/公历切换） |
| 出生时间 | 时辰（23:00-01:00 子时等） |
| 性别 | 男/女（影响大运方向） |
| 时区 | 自动获取，可手动选择 |
| 出生地点 | 可选，影响真太阳时 |

### 2.2 命盘展示（核心动画）

#### 2.2.1 环形结构
```
最外圈: 流年（当前年份高亮）
第二圈: 大运（每10年一换）
第三圈: 地支（12宫位）
第四圈: 天干
中心:   日主（命主）
```

#### 2.2.2 动画效果
| 阶段 | 动画描述 | 时长 |
|------|---------|------|
| 启动动画 | 环形从中心向外展开 | 2s |
| 自转动画 | 各环以不同速度自转 | 持续 |
| 落位动画 | 天干地支落入正确位置 | 1s |
| 结果展示 | 最终命格文字浮现 | 1s |

#### 2.2.3 命盘交互
- 点击任意宫位显示详情
- 长按暂停/播放动画
- 手势缩放命盘

### 2.3 八字排盘算法

#### 2.3.1 四柱计算
```
年柱: 60甲子（年尾数→天干 + 向前/后推地支）
月柱: 五虎遁（年干→月干）+ 节气（决定月令）
日柱: 儒略日精确计算（日干支）
时柱: 五鼠遁（日干→时干）
```

### 2.4 节气计算（关键）

#### 2.4.1 二十四节气
```
春季: 立春 → 雨水 → 惊蛰 → 春分 → 清明 → 谷雨
夏季: 立夏 → 小满 → 芒种 → 夏至 → 小暑 → 大暑
秋季: 立秋 → 处暑 → 白露 → 秋分 → 寒露 → 霜降
冬季: 立冬 → 小雪 → 大雪 → 冬至 → 小寒 → 大寒
```

#### 2.4.2 节气计算算法
- 使用**太阳黄经**精确计算
- 当太阳黄经到达 315° 时为立春，330° 为雨水，以此类推
- 需要天文数据或近似公式

### 2.5 大运排法

#### 2.5.1 顺逆规则
| 年干阴阳 | 性别 | 大运方向 |
|---------|------|---------|
| 阳年 | 男 | 顺行 |
| 阳年 | 女 | 逆行 |
| 阴年 | 男 | 逆行 |
| 阴年 | 女 | 顺行 |

#### 2.5.2 起运年龄计算
```
规则: 3日 = 1岁，1日 = 4个月，1时辰 = 10天
步骤:
1. 计算出生到下一个节气的天数
2. 转换为岁数（除以3）
3. 确定大运起始年龄
```

### 3.3 命盘动画实现方案

#### 方案A: Canvas（推荐）
- 性能好，适合复杂动画
- 便于实现环形旋转和粒子效果
- 需要自己处理渲染循环

#### 方案B: SVG + Framer Motion
- 开发效率高
- 适合不太复杂的动画
- 性能略差

## 5. 动画详细设计

### 5.1 启动动画流程
```
0ms   - 屏幕中央出现光点
500ms - 光点扩散为中心圆（日主）
1000ms- 外圈展开（地支）
1500ms- 第二圈展开（天干）
2000ms- 第三圈展开（大运）
2500ms- 最外圈展开（流年）
3000ms- 所有环开始自转
4000ms- 命格文字浮现
```

### 5.2 自转速度
| 层级 | 速度 | 方向 |
|------|------|------|
| 流年环 | 慢 | 顺时针 |
| 大运环 | 中 | 顺时针 |
| 地支环 | 快 | 逆时针 |
| 天干环 | 最快 | 逆时针 |
| 中心 | 静止 | - |

## 8. 备注

### 8.1 精度说明
- 节气时间精确到**分钟**为宜
- 使用近似算法可能有几分钟误差
- 可接入专业天文数据提高精度

### 8.2 简化方案
第一版可使用简化的平均节气日期，后续升级为精确计算。
