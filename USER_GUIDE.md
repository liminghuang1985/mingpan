# 命盘App使用指南

## 📱 快速开始

### 1. 运行项目

```bash
# 安装依赖
flutter pub get

# 运行到设备（自动检测）
flutter run

# 运行到特定平台
flutter run -d chrome        # Web
flutter run -d android       # Android
flutter run -d ios          # iOS
flutter run -d windows      # Windows
flutter run -d macos        # macOS
flutter run -d linux        # Linux
```

### 2. 首次使用

#### 2.1 输入出生信息
1. **出生日期**: 点击"出生日期"卡片，选择年月日
2. **出生时间**: 点击"出生时间"卡片，选择时分
3. **性别**: 选择"男"或"女"（影响大运顺逆）

#### 2.2 排盘
- 点击"排盘"按钮
- 稍等片刻，命盘将自动生成并展示

#### 2.3 查看命盘
- **命盘结构**（从外到内）：
  - 最外圈：流年环（当前年份高亮）
  - 第二圈：大运环（10年一换）
  - 第三圈：地支环（12宫位）
  - 第四圈：天干环
  - 中心圆：日主（命主五行）

## 🎮 交互功能

### 命盘交互

#### 动画控制
- **播放/暂停**: 点击右上角▶️/⏸️图标
- **长按暂停**: 长按命盘任意位置暂停/恢复动画
- **手势缩放**: 双指缩放命盘大小（0.5x - 3.0x）

#### 点击查看详情
- **点击宫位**: 查看地支详细信息
- **点击日主**: 查看日主信息
- **点击大运环**: 查看当前大运
- **点击流年环**: 查看当前流年

### 详情页功能

#### 进入详情页
点击"查看详情"按钮，进入全面的命盘分析页面。

#### 详情页内容

##### 1️⃣ 四柱总览
- 年柱、月柱、日柱、时柱
- 日主天干及五行

##### 2️⃣ 大运时间线 ⭐
- **横向滚动**: 左右滑动查看10步大运
- **当前大运**: 蓝色高亮 + 进度条
- **点击大运**: 弹出详情对话框
  - 大运干支
  - 年龄段（XX-XX岁）
  - 年份范围
  - 顺/逆行标记

##### 3️⃣ 五行能量分析 ⭐
- **能量条**: 5个进度条展示木火土金水强弱
- **旺衰等级**: 极旺/偏旺/中和/偏弱/极弱
- **最旺/最弱**: 自动分析并给出建议
- **相生相克**: 五行关系图解

##### 4️⃣ 地支藏干
- 年支、月支、日支、时支藏干详情
- 本气/中气/余气标注

##### 5️⃣ 十神分析 ⭐
- **柱位十神**: 年/月/时柱的天干地支十神
- **十神颜色**: 不同十神不同颜色编码
- **十神统计**: 出现次数统计

##### 6️⃣ 流年轮播 ⭐
- **左右滑动**: 查看前后年份流年
- **流年卡片**: 
  - 流年干支
  - 流年概要
  - 冲合关系（⚡/❤️标签）
  - 流月网格（12个月）
- **当前年份**: 金色高亮卡片

##### 7️⃣ 流年网格（12年） ⭐
- **年历视图**: 3×4网格展示12年
- **点击查看**: 弹出流年详情
- **当前年份**: 高亮标记

## 💡 使用技巧

### 1. 命盘查看技巧

- **滚动查看**: 从上到下依次查看各项分析

### 3. 大运时间线
- **左右滑动**: 查看过去和未来的大运
- **查看进度**: 当前大运显示进度条（X/10年）
- **点击详情**: 点击任意大运查看详细信息

### 4. 流年查看
- **轮播模式**: 左右滑动查看前后年份
- **网格模式**: 12年一览，快速定位
- **流月展示**: 展开流年卡片查看12个月详情

### 5. 五行分析
- **能量条**: 直观对比五行强弱
- **旺衰等级**: 快速判断五行平衡
- **用神建议**: 根据最弱五行补充

## 🔍 常见问题

### Q1: 命盘不显示？
**A**: 检查是否已点击"排盘"按钮，确保输入了完整的出生信息。

### Q2: 动画卡顿？
**A**: 
- 关闭动画（点击暂停按钮）
- 在设置中降低动画质量
- 检查设备性能

### Q3: 八字准确吗？
**A**: 
- 算法基于传统命理规则，结果准确
- 节气使用简化公式，可能有几分钟误差
- 建议专业命理师最终确认

### Q4: 如何保存命盘？
**A**: 
- 当前版本通过截图保存
- 未来版本将支持命盘存档功能

### Q5: 不同平台显示不一致？
**A**: 
- 已实现响应式设计，自动适配
- 命盘尺寸会根据屏幕大小调整
- 功能完全一致

### Q6: 能看历史人物的命盘吗？
**A**: 可以！选择历史人物的出生日期即可。注意：
- 年份范围：1900-2100
- 需要确切的出生时间
- 节气以公历为准

## 🎓 命理基础知识

### 四柱含义
- **年柱**: 代表祖辈、父母，影响1-16岁
- **月柱**: 代表兄弟姐妹、父母，影响17-32岁
- **日柱**: 代表自己、配偶，影响33-48岁
- **时柱**: 代表子女、晚年，影响49岁以后

### 大运规律
- **起运**: 出生后X岁Y月开始
- **每步**: 10年一换
- **顺逆**: 由年干阴阳和性别决定
  - 阳男阴女：顺行
  - 阴男阳女：逆行

### 流年流月
- **流年**: 每年的运势，以当年干支为准
- **流月**: 每月的运势，以当月干支为准
- **冲合**: 与命局的冲突或和谐关系

### 五行强弱
- **极旺**: 该五行过强，需克泄
- **偏旺**: 该五行较强
- **中和**: 最佳平衡状态
- **偏弱**: 该五行较弱，需生扶
- **极弱**: 该五行过弱，需大力补充

### 十神作用
- **比肩**: 兄弟、竞争、平等
- **劫财**: 合作、争夺
- **食神**: 表达、才华、口福
- **伤官**: 创新、反叛、技艺
- **偏财**: 意外之财、投资
- **正财**: 工薪、固定收入
- **七杀**: 权力、压力、小人
- **正官**: 职位、名声、约束
- **偏印**: 偏门学问、继母
- **正印**: 学历、文凭、母亲
